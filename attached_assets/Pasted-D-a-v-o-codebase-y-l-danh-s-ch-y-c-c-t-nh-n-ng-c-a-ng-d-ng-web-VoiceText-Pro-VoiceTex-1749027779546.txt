Dựa vào codebase, đây là danh sách đầy đủ các tính năng của ứng dụng web VoiceText Pro:

🎵 VoiceText Pro - Ứng dụng Web Text-to-Speech
🏠 Trang chủ (Landing Page)
Giao diện hiện đại với dark theme
Giới thiệu tính năng chính
Call-to-action để bắt đầu sử dụng
🎙️ Tính năng Text-to-Speech chính
Nhập văn bản: Ô text area lớn để nhập nội dung
Chọn giọng nói: Dropdown với nhiều giọng khác nhau
Điều chỉnh tốc độ: Slider để thay đổi tốc độ đọc
Generate Audio: Tạo file âm thanh từ văn bản
Play/Pause: Phát và tạm dừng audio
Download: Tải file âm thanh về máy
🔊 Hệ thống giọng nói
OpenAI Voices: Alloy, Echo, Fable, Onyx, Nova, Shimmer
Chất lượng cao: Giọng tự nhiên, rõ ràng
Hỗ trợ đa ngôn ngữ: Tiếng Anh và tiếng Việt
📁 Quản lý file audio
Lịch sử audio: Xem tất cả file đã tạo
Metadata: Hiển thị thông tin chi tiết file
Storage: Lưu trữ và quản lý file audio
📋 Hệ thống Templates
Templates có sẵn: Các mẫu văn bản thông dụng
Quick start: Nhanh chóng tạo audio từ template
Danh mục: Phân loại theo chủ đề
⚙️ Cài đặt ứng dụng
Free limits: Giới hạn file và ký tự cho user miễn phí
Configuration: Tùy chỉnh các thông số hệ thống
User settings: Cài đặt cá nhân
🔔 Hệ thống thông báo
Real-time notifications: Thông báo thời gian thực
Unread counter: Đếm số thông báo chưa đọc
Notification history: Lịch sử thông báo
